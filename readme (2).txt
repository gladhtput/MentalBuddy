Untuk cek database via Visual Studio diperlukan untuk menginstall access database engine 2007.

Kekurangan aplikasi:
1. General Discussion tak berfungsi sebagai mana mestinya karena kesalahpahaman pembuat ide dan coder utama sehingga
   hasilnya lebih seperti artikel daripada general discussion.
2. Tidak adanya register seharusnya dapat diganti dengan penambahan akun admin agar jumlah psikiater atau psikolog
   dapat bertambah.
3. Tidak diadakannya register karena bisa dipastikan akan terjadi fake account untuk psikolog ataupun psikiater.
